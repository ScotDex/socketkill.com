Mission Profile

Socket.Kill is a high-performance front end inspired by the retro-futuristic CRT monitor presentation from the Alien franchise. Optimized for minimal overhead, a centralized image proxy and localized cache keep parsing performance under 100ms.

Latency

Speeds governed by RedisQ API pulse rates.

Telemetry

Losses >1B ISK trigger RED. Extremes in GOLD.

Privacy

No pilot data stored. Localized cache for speed only.
